Inspired by the late 90s Computer companion Bonzi Buddy.

GoobyDuck is a computer companion set to help you, and entertain.

GoobyDuck has a tasks bar where he can do tasks for you, and he has a game bar where you can play games.

Games:  Duck Clicker, Duck Dodge, Duck pong, and Duck Car. Gooby is also the most annoying companion, and moves randomly on your screen at a random time. NOTE: Please note that this Companion is still in prototype phrase, and still will be updated and many other features will be added (Helpful, more settings, more games)
